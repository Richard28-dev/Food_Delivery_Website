# Food_Delivery_Website
# 🍔 Food Delivery Website

A full-stack Food Delivery Web Application developed using the MERN Stack. This project allows users to browse food items, add products to cart, place orders, and manage deliveries through a responsive and user-friendly interface.

---

## 🚀 Features

* User Authentication & Authorization
* Browse Food Categories and Items
* Search and Filter Functionality
* Add to Cart & Remove from Cart
* Online Food Ordering System
* Order Management
* Responsive UI Design
* Admin Panel for Food Management
* Secure Backend APIs
* Database Integration with MongoDB

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/food-delivery-website.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd food-delivery-website
```

### 3️⃣ Install Dependencies

#### Frontend

```bash
cd frontend
npm install
```

#### Backend

```bash
cd backend
npm install
```

---

## ▶️ Run the Project

### Start Backend Server

```bash
npm start
```

### Start Frontend

```bash
npm run dev
```
